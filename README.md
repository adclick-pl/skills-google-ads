# Skile Google Ads

Otwarta kolekcja **skilli Claude** do pracy z Google Ads — po polsku, pod realne workflowy agencji i specjalistów PPC.

Skille działają zarówno w **Claude** (web/desktop) jako wgrane skille, jak i w **Claude Code** jako pliki w `.claude/skills/`.

## Dostępne skille

| Skill | Co robi |
|---|---|
| [`gads-reklamy`](gads-reklamy/SKILL.md) | Pisze reklamy Google Ads (RSA) po polsku w 4-etapowym procesie: research firmy i konkurencji → kąty reklamowe → akceptacja → 20 nagłówków + 6 tekstów + objaśnienia/sitelinki. |

Więcej skili w drodze.

## Jak użyć

### W Claude (web / desktop)
1. Pobierz plik `SKILL.md` wybranego skilla.
2. W Claude → ustawienia skilli → dodaj nowy skill → wklej zawartość.
3. Wywołaj w czacie zgodnie z opisem (np. *"napisz reklamy Google Ads dla strony https://..."*).

### W Claude Code
1. Skopiuj folder skilla (np. `gads-reklamy/`) do `.claude/skills/` w swoim projekcie.
2. Skill jest dostępny automatycznie — Claude Code uruchomi go gdy zapytanie pasuje do opisu.
